# BulMark
Marketplace template build with Bulma CSS

## Installation


## Building yourself

1. Clone the project
```sh
  git clone https://link-to-project
```

2. Install dependencies
```sh
  npm install
```

3. Compile Template CSS
```sh
  npm run start
```
