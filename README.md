# BulMark
Marketplace template build with Bulma CSS

## Installation


## Building yourself

1. Clone the project
```sh
  git clone https://github.com/mocfaisal/bulmark.git
```

2. Install dependencies
```sh
  npm install
```

3. Compile Template CSS
```sh
  npm run start
```
